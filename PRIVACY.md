# Privacy Policy — Prompt Pal

_Last updated: May 2026_

## Data storage

Prompt Pal stores your prompts locally on your device using `chrome.storage.local`. This data never leaves your browser and is not transmitted to any server.

## GitHub sync

If you use the optional GitHub sync feature, the extension fetches `.md` files from the public GitHub repository URL you provide. This request is made directly from your browser to the GitHub API — no data passes through any third-party server operated by this extension. No personal information is sent as part of this request.

## No tracking or analytics

Prompt Pal collects no usage data, no analytics, and no personal information of any kind.

## Contact

If you have questions, open an issue at the GitHub repository for this project.
